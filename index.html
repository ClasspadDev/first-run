Final Setup Steps

Complete these steps to get Hollyhock running on your ClassPad

1

Download the Launcher File

First, download the essential Hollyhock launcher file to your computer.

  • Download run.bin from the link below
  • The file will be saved as run.bin automatically
Download run.bin
2

Transfer Files to Your Calculator

Copy the downloaded files to your ClassPad II:

  • Connect your ClassPad II to your computer via USB
  • Copy run.bin to the root directory of your calculator
  • Copy any HH3 programs and games to the root directory as well (see below)
  • Safely eject your calculator from your computer
3

Launch Hollyhock

Access the Hollyhock launcher from your ClassPad's menu:

  • Go to the main menu
  • Navigate to the "System" menu on the second page
  • Select "Hollyhock Launcher" from the system menu in the top left

Troubleshooting

Calculator Freezes

If your calculator freezes when launching Hollyhock, check that run.bin was copied correctly to the root directory. Try re-downloading and re-copying the file.

Nothing Displays in Launcher

If no programs appear in the launcher, verify that you have copied HH3 programs or games to the root directory of your calculator. The launcher needs these files to display options.

File Transfer Issues

If you're having trouble transferring files, make sure your calculator is properly connected and recognized by your computer. Try using a different USB cable if necessary.
